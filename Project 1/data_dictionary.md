# Project 1 — Data Dictionary

All tables live in the `airline_revenue` PostgreSQL database (schema: `public`).
Timestamps are stored as `TIMESTAMPTZ` in **UTC (Zulu)**.
Distance is always in **statute miles**.

---

## dim_time

Calendar dimension used for time-series joins.

| Column | Type | Description |
|--------|------|-------------|
| time_id | DATE PK | Calendar date |
| year | SMALLINT | 4-digit year |
| quarter | SMALLINT | 1–4 |
| month | SMALLINT | 1–12 |
| month_name | VARCHAR | e.g. "January" |
| week_of_year | SMALLINT | ISO week number (1–53) |
| day_of_week | SMALLINT | 0 = Monday … 6 = Sunday (ISO) |
| day_name | VARCHAR | e.g. "Monday" |
| is_weekend | BOOLEAN | TRUE for Saturday/Sunday |
| is_holiday | BOOLEAN | TRUE for U.S. federal holidays |

**Source:** Generated by `transform.build_dim_time()` from flight dates.

---

## aircraft_configs

One row per United aircraft variant × cabin class combination.

| Column | Type | Description |
|--------|------|-------------|
| config_id | SERIAL PK | Surrogate key |
| aircraft_variant | VARCHAR | United marketing name (e.g. "B777-300ER", "737-800 Version 1") |
| fa_aircraft_codes | TEXT[] | FlightAware ICAO type codes that map to this variant |
| cabin_class | VARCHAR | United cabin name (e.g. "United Polaris® Business Class") |
| cabin_tier | VARCHAR | Normalised tier: `first`, `business`, `premium_plus`, `economy_plus`, `economy` |
| seat_count | SMALLINT | Total seats in this cabin on this aircraft |
| seat_configuration | VARCHAR | Seat layout string (e.g. "1–2–1", "3-3-3") |
| seat_pitch_in | NUMERIC | Seat pitch, inches (first numeric value from source) |
| seat_width_in | NUMERIC | Seat width, inches |
| has_wifi | BOOLEAN | Wi-Fi available |
| has_power | BOOLEAN | AC / USB power outlet available |

**Source:** `Project 1/aircraft_data/UAL_aircraft_data.json` (United fleet page).

## flights

One row per operated or scheduled flight leg.

| Column | Type | Description |
|--------|------|-------------|
| flight_id | VARCHAR PK | FlightAware `fa_flight_id` (globally unique) |
| ident | VARCHAR | Callsign (e.g. "UAL288") |
| ident_iata | VARCHAR | IATA flight number (e.g. "UA288") |
| operator_icao | CHAR(3) | Airline ICAO code (e.g. "UAL") |
| operator_iata | CHAR(2) | Airline IATA code (e.g. "UA") |
| flight_number | INTEGER | Numeric flight number |
| registration | VARCHAR | Tail number (N-number) |
| aircraft_type | VARCHAR | FlightAware ICAO aircraft type code (e.g. "B77W") |
| aircraft_variant | VARCHAR | Mapped United seat-map variant; FK → aircraft_configs |
| origin_iata | CHAR(3) FK | Origin airport |
| destination_iata | CHAR(3) FK | Destination airport |
| route_distance_mi | INTEGER | Great-circle route distance (miles) from FlightAware |
| scheduled_out | TIMESTAMPTZ | Scheduled gate pushback (UTC) |
| scheduled_off | TIMESTAMPTZ | Scheduled wheels-off (UTC) |
| scheduled_on | TIMESTAMPTZ | Scheduled wheels-on (UTC) |
| scheduled_in | TIMESTAMPTZ | Scheduled gate arrival (UTC) |
| actual_out | TIMESTAMPTZ | Actual gate pushback (UTC) |
| actual_off | TIMESTAMPTZ | Actual wheels-off (UTC) |
| actual_on | TIMESTAMPTZ | Actual wheels-on (UTC) |
| actual_in | TIMESTAMPTZ | Actual gate arrival (UTC) |
| block_time_min | INTEGER (computed) | Block time in minutes (actual_on − actual_off; falls back to scheduled) |
| flight_date | DATE (computed) | UTC date of departure (actual_off ?? scheduled_off) |
| departure_delay_min | INTEGER | Minutes of departure delay (negative = early) |
| arrival_delay_min | INTEGER | Minutes of arrival delay |
| status | VARCHAR | FlightAware status string (e.g. "Scheduled", "En Route") |
| cancelled | BOOLEAN | Flight was cancelled |
| diverted | BOOLEAN | Flight was diverted |
| fa_seats_first | SMALLINT | Seats in First as reported by FA (sparse) |
| fa_seats_business | SMALLINT | Seats in Business as reported by FA (sparse) |
| fa_seats_coach | SMALLINT | Seats in Coach as reported by FA (sparse) |
| data_source | VARCHAR | Always "flightaware" |
| ingested_at | TIMESTAMPTZ | Row insertion timestamp |

**Source:** FlightAware AeroAPI `/operators/{id}/flights`. Pulled daily.

## flight_capacity

Exploded flight × cabin table for Available Seat Mile (ASM) calculations.

| Column | Type | Description |
|--------|------|-------------|
| id | BIGSERIAL PK | Surrogate key |
| flight_id | VARCHAR FK | References flights |
| aircraft_variant | VARCHAR | United seat-map variant |
| cabin_class | VARCHAR | United cabin name |
| cabin_tier | VARCHAR | Normalised tier (see aircraft_configs) |
| seats | SMALLINT | Seats in this cabin on this flight |
| route_distance_mi | INTEGER | Route distance (miles) |
| asm | INTEGER (computed) | Available Seat Miles = seats × route_distance_mi |

**Derivation:** `transform.compute_flight_capacity()` joins `flights.aircraft_variant` → `aircraft_configs`.

---

## fuel_prices

Weekly U.S. Gulf Coast Jet-A spot prices.

| Column | Type | Description |
|--------|------|-------------|
| price_date | DATE PK | Friday of the reporting week |
| jet_a_usd_per_gal | NUMERIC(6,4) | Price in USD/gallon |
| data_source | VARCHAR | Always "eia" |
| ingested_at | TIMESTAMPTZ | Row insertion timestamp |

**Source:** U.S. EIA API v2, series `PET.EER_EPJK_PF4_RGC_DPG.W` — *U.S. Gulf Coast Kerosene-Type Jet Fuel, Weekly*.

**Matching:** `v_fuel_matched_flights` attaches the most-recent price ≤ `flight_date` via LATERAL join.

## aircraft_fuel_burn

Cruise-phase fuel burn rates by aircraft variant.

| Column | Type | Description |
|--------|------|-------------|
| fuel_burn_id | SERIAL PK | Surrogate key |
| aircraft_variant | VARCHAR UNIQUE | United seat-map name |
| gph_cruise | NUMERIC | Gallons per hour at cruise |
| gph_source | VARCHAR | Source: "POH", "BTS_P12a", "estimated" |
| notes | TEXT | Caveats / assumptions |

**Sources:** Pilot Operating Handbooks; cross-validated against BTS Form 41 Schedule P-12(a) reported fuel per block hour.

---

## bts_t100_segments

Route-level monthly traffic from BTS T-100 Segment data.

| Column | Type | Description |
|--------|------|-------------|
| id | BIGSERIAL PK | Surrogate key |
| report_period | DATE | First day of the report month (e.g. 2024-01-01) |
| carrier_code | CHAR(2) | IATA carrier code |
| carrier_name | VARCHAR | Airline name |
| aircraft_type_bts | VARCHAR | BTS aircraft type code |
| origin_iata | CHAR(3) | Origin airport |
| destination_iata | CHAR(3) | Destination airport |
| departures_performed | INTEGER | Actual departures that month |
| seats_available | INTEGER | Total available seats that month |
| passengers | INTEGER | Revenue passengers carried |
| load_factor | NUMERIC (computed) | passengers / seats_available |
| payload_lbs | NUMERIC | Total payload (lbs) |
| distance_mi | INTEGER | Stage distance (miles) |
| asm | BIGINT (computed) | Available Seat Miles = seats_available × distance_mi |
| rpm | BIGINT (computed) | Revenue Passenger Miles = passengers × distance_mi |

**Source:** BTS T-100 Domestic and International Segment, downloaded quarterly.
**Download:** https://www.transtats.bts.gov (filter: Carrier = UA, DL, AA, WN, AS, B6).

---

## bts_db1b_fares

Route-level average fares from the BTS DB1B 10% ticket sample.

| Column | Type | Description |
|--------|------|-------------|
| id | BIGSERIAL PK | Surrogate key |
| report_quarter | DATE | First day of the quarter (e.g. 2024-01-01) |
| carrier_code | CHAR(2) | IATA carrier code |
| origin_iata | CHAR(3) | Origin market |
| destination_iata | CHAR(3) | Destination market |
| cabin_class_bts | VARCHAR | BTS cabin label: Coach, Business, First, Premium |
| avg_fare_usd | NUMERIC(8,2) | Average itinerary fare (USD, aggregated from DB1B) |
| passengers | INTEGER | Sampled passengers (×10 to estimate population) |
| miles | INTEGER | Average itinerary distance |
| yield_per_mile | NUMERIC (computed) | avg_fare_usd / miles |

**Source:** BTS DB1B Coupon and Market datasets, downloaded quarterly.

## Views

### v_flight_summary
Flight-level rollup: total seats, total ASM, premium vs. economy breakdown.

### v_fuel_matched_flights
Extends `flights` with the nearest weekly fuel price and estimated fuel cost:
`est_fuel_cost_usd = (block_time_min / 60) × gph_cruise × jet_a_usd_per_gal`
